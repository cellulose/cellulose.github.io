Resources.Disk

  • Summary
  • Types
  • Functions

Uses linux command line utility df to get disk information.

Can either gather information for all disks or filter disks by providing :disks option to info/1 method.

Source

info(args \\ [])

Returns Dict of disk usage from ‘df’ or an error

Specs:

  • info(t) :: t | {:error, :df_not_available}

Returns Dict of disk usage from ‘df’ or an error

Options

The :disks options may be passed with a list of keys and mount points to filter on
